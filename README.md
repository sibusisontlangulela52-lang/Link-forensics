# Link-forensics
Browser-based phishing URL risk scanner built with HTML, CSS and JavaScript
Link Forensics

Link Forensics is a browser-based phishing URL risk scanner built with HTML, CSS, and JavaScript. It analyzes suspicious links using client-side heuristics and highlights phishing-related indicators such as suspicious top-level domains, shortened URLs, raw IP hosts, punycode domains, excessive subdomains, brand impersonation patterns, non-standard ports, and credential-related path terms.

Features

- URL risk scoring system
- Low / medium / high severity verdicts
- URL anatomy breakdown
- Decoded URL inspection
- Registrable domain analysis
- Brand impersonation detection
- Suspicious TLD and subdomain checks
- Detection of IP-based hosts, punycode, and disguised extensions

Tech Stack

- HTML
- CSS
- JavaScript

Purpose

This project was built as a cybersecurity portfolio project to demonstrate practical skills in phishing detection concepts, frontend development, and security-focused tool design.
